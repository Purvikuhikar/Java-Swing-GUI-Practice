# 🧾 Salary Calculator (Java Swing)

## 📖 Introduction
This application calculates employee salary including DA, HRA, gross salary, and net salary.

---

## ⚙ How it Works

1. Enter employee details
2. Enter salary components
3. Program calculates allowances
4. Displays full salary breakdown

---

## ✨ Features

- Dynamic salary calculation
- Structured output display
- Handles multiple inputs
- Holds temporary data

---

## 💼 Salary Components

- Basic Salary
- DA (Dearness Allowance)
- HRA (House Rent Allowance)
- Medical
- Conveyance

---

## 📚 Concepts Used

- Conditional logic
- Arithmetic calculations
- GUI forms

---

## 🎯 Purpose

This project simulates a real-world employee salary system.
