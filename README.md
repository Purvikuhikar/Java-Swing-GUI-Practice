# Java-Swing-GUI-Practice-
Beginner Java GUI interfaces created using Java Swing while learning programming concepts. These small applications demonstrate input handling, event-driven programming, and basic logic implementation.

## Purpose
The goal of these programs is to practice building graphical user interfaces and implementing basic programming logic.

## Technologies Used
- Java
- Java Swing
- NetBeans IDE

## Programs Included
- Basic calculator
- Even and Odd Checker

These mini applications focus on understanding GUI components, event handling, and program logic.
