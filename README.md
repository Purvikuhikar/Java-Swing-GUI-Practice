# Java Swing GUI Practice Programs

## Introduction

This repository contains a collection of small graphical user interface (GUI) programs developed using Java Swing.  
These programs were created as part of my learning process while studying Java programming and exploring how graphical applications work.

Each program focuses on implementing basic programming concepts such as user input handling, conditional statements, arithmetic operations, and event-driven programming.

The goal of these projects is to practice building interactive desktop applications and to understand how Java Swing components work together to create functional interfaces.

---

## What These Mini Interfaces Are

The programs in this repository are small GUI-based applications designed to perform specific tasks. Each interface allows the user to enter input, perform an operation, and view the result through graphical components.

Some examples of programs included in this repository are:

- Basic Calculator :- Performs addition, subtraction, multiplication, and division 
- Even and Odd Checker :- Determines whether a number is even or odd , Positive or Negative

These programs demonstrate how simple programming logic can be combined with graphical elements to build interactive applications.

## Features of the Programs

- Graphical user interface using Java Swing
- Input fields for user data

- Output displayed through labels, text areas, or dialog boxes

---

## How to Run the Project

1. Download or clone the repository.
2. Open the project in NetBeans IDE.
3. Navigate to the `src` folder.
4. Open the desired Java file containing the GUI program.
5. Run the file to launch the interface.

The program will open a graphical window where the user can enter input and interact with the application.
