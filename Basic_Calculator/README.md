# Basic Calculator (Java Swing)

This project is a simple calculator application built using Java Swing.  
The program allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division through a graphical user interface.

## How it Works

1. The user enters two numbers in the input fields.
2. The user selects an operation button:
   - Addition (+)
   - Subtraction (-)
   - Multiplication (*)
   - Division (/)
3. When an operation button is clicked, the program performs the selected calculation.
4. The result is displayed in the result text field.

## Features

- Simple and user-friendly GUI
- Performs four basic arithmetic operations
- Uses event handling for button actions
- Displays results instantly in the interface

## Technologies Used

- Java
- Java Swing (GUI components)
- NetBeans IDE

## Purpose

This project was created as a practice exercise to understand:
- GUI design using Java Swing
- Event handling in Java
- Arithmetic operations in programming
- User input and output handling
